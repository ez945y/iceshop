# 刨冰人格測驗

挑出你必加、與絕對不加的配料，測出你的刨冰人格。

還能輸入另一半的代碼，看看你們一起吃這碗冰有多合拍。不合也沒關係，一人一碗，各自美麗。

點開就玩，把結果分享給朋友吧。
